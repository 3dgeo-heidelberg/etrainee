# EnMAP-Box

EnMAP-box is a QGIS plugin for working with imaging spectroscopy remote sensing data:  
[https://www.enmap.org/data_tools/enmapbox/](https://www.enmap.org/data_tools/enmapbox/)
  
It is a free and open source software released under the GNU General Public License.

## Download and installation

Follow the instructions for installing the EnMAP-box QGIS plugin [Manual for installing the plugin](https://enmap-box.readthedocs.io/en/latest/usr_section/usr_installation.html)  

You need to install QGIS and required Python packages first.

## Getting started / external material

Introduction to the EnMAP-box and how to handle spectral Iibraries can be found on the [EnMAP-box
workshop tutorial website](https://enmap-box.readthedocs.io/en/latest/usr_section/workshop_tutorials/index.html). Description of the specific software functions can be found in the [User
Manual](https://enmap-box.readthedocs.io/en/latest/usr_section/usr_manual/usr_manual.html).
